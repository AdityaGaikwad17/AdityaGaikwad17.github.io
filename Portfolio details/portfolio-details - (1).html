Bachelor of Engineering Project

We Build a Machine Learning model to detect Cyber Attacks in a network. The model is trained on a dataset containing various types of network traffic, both normal and malicious. We used different algorithms to train the model and evaluated their performance using metrics such as accuracy, precision, recall, and F1-score. The best-performing model was then deployed in a simulated network environment to test its effectiveness in real-time attack detection.
